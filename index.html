Raster processing suite

A raster processing plugin for QGIS

Download this project as a .zip file Download this project as a tar.gz file

Raster Processing Suite provides users with a graphical interface to PyRaster functionality within the QGIS environment. Using the plugin users can open spatial image data, perform raster mathematical operations and create new scenes from the results. Raster processor can also generate on-the-fly Python image processing code for users to develop more advanced tasks such as batch processing.
